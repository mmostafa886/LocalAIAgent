┌──────────────────────────────────────────────────────────────┐
│ 🐛 TROUBLESHOOTING                                           │
└──────────────────────────────────────────────────────────────┘

PROBLEM: Logs say "Cannot connect to Ollama"
SOLUTION: Start Ollama service
          ollama serve

PROBLEM: Logs say "Model not found"
SOLUTION: Pull your model
          ollama pull llama3.2-vision:11b

PROBLEM: No logs appearing
SOLUTION: Check browser console (F12)
          Refresh page and try again

PROBLEM: Generation seems stuck
SOLUTION: Check logs for last message
          Wait up to 2 minutes for complex stories
          Look for [ERROR] messages

PROBLEM: Red error in logs
SOLUTION: Read the error message
          Check Ollama is running
          Verify model is available
          Check acceptance criteria are valid
